# Coda

O Coda surgiu da necessidade de ter uma ferramenta que me ajudasse a ter uma melhor visão das minhas finanças pessoais.

Inicialmente, a stack que pretendia utilizar era React + ElysiaJS, porém acabei optando por utilizar Laravel no backend por conta da matéria na faculdade.

## Funcionalidades

As funcionalidades que serão implementadas no Coda são 100% voltadas as minhas necessidades.

[] Dashboard simples para acompanhamento
[] Extrato para acompanhamento de receitas e despesas
[] Gestão de assinaturas e despesas recorrentes
[] Cadastro de conta e cartões
[] Categorias de receita e despesa aninhadas
